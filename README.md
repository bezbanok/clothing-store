# Интернет-магазин одежды

Проект представляет собой API интернет-магазина одежды.

## Основные функции

- просмотр каталога товаров;
- фильтрация товаров по категории, размеру и цвету;
- добавление товаров в корзину;
- оформление заказа;
- регистрация пользователя;
- просмотр профиля пользователя.

## Стек технологий

- Python;
- FastAPI;
- Pydantic;
- Scalar;
- Uvicorn.
